# Network Engineering Lab Portfolio – Fanyana Masango

This repository contains a structured set of networking labs I built to refresh
and deepen my practical networking skills (CCNA–CCNP level).

## Objectives
- Reinforce routing and switching fundamentals
- Practice real-world enterprise scenarios
- Build troubleshooting intuition
- Maintain a personal knowledge base

## Lab Themes
- Switching (VLANs, STP, EtherChannel)
- Routing (Static, OSPF, EIGRP, BGP)
- IPv6
- VRF-Lite
- NAT, DHCP, HSRP
- VPNs and WAN failover
- Security basics (ACLs, port security)

## Structure
Labs are organized by topic. Each lab contains:
- A topology diagram
- Device configurations
- Verification outputs
- Lessons learned

## Tools Used
- Cisco Packet Tracer
- EVE-NG

## About Me
Network & Telecommunications Engineer with 3+ years experience in routing, switching,
wireless, and VoIP. CCNA certified. Building toward CCNP.
